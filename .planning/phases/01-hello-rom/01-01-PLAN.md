  <action>
    Create `build.zig` following the castholm/zig-examples/breakout pattern:

    1. Import `std` and `translate_c`.
    2. Accept `target` via `b.standardTargetOptions(.{})` and `optimize` via `b.standardOptimizeOption(.{})`.
    3. SDL3 dependency: `b.dependency("sdl", .{
         .target = target,
         .optimize = optimize,
         .preferred_linkage = .static,
         .strip = false,  // Debug mode; release sets strip = true
       })`. Get artifact: `sdl_dep.artifact("SDL3")`.
    4. translate-c dependency: `b.dependency("translate_c", .{})`. Create `Translator` with a `c.h` file containing:
       - `#define SDL_DISABLE_OLD_NAMES`
       - `#include <SDL3/SDL.h>`
       - `#define SDL_MAIN_HANDLED`
       - `#include <SDL3/SDL_main.h>`
       Call `translator.linkLibrary(sdl_lib)`.
    5. Executable `zigboy`: root_source_file = `src/main.zig`. Add import "c" via translator.mod. Link sdl_lib. Link libC.
    6. Set `exe.root_module.strip = true` and `exe.root_module.lto = .full` when optimize != .Debug (per D-04).
    7. Install step via `b.installArtifact(exe)`.
    8. Run step: `b.addRunArtifact(exe)`, forward `b.args`, create step "run".
    9. Test target: create `const test_exe = b.addTest(.{ .root_source_file = b.path("src/main.zig"), .target = target, .optimize = optimize })` and a "test" step that runs it. The test target is a placeholder; Plan 01-03 replaces it with the real Blargg test.
    10. Use `b.writeFiles()` for the C wrapper header (not a physical file). Follow the castholm/breakout pattern exactly.

    - Import `std`, import `c` (SDL3 via translate-c), import `lib` (lib.zig).
    - `pub fn main() !void`:
      - Parse allocator: `var gpa = std.heap.GeneralPurposeAllocator(.{}){}; defer _ = gpa.deinit(); const allocator = gpa.allocator();`
      - Get args: `const args = try std.process.argsAlloc(allocator); defer std.process.argsFree(allocator, args);`
      - If args.len < 2: print usage "Usage: zigboy <rom-path>" to stderr, exit with error.
      - `const rom_path = args[1]` — store the ROM path string.
      - Per D-02: Call `c.SDL_Init(c.SDL_INIT_VIDEO)`. If it returns false, print SDL error and return error. Use `c.SDL_GetError()` for error string.
      - Print: `std.debug.print("ROM path: {s}\n", .{rom_path});` and `std.debug.print("SDL_Init OK\n", .{});`
      - Call `c.SDL_Quit();` via defer or at end.
      - Note: Emulator.init is NOT called yet — this plan only proves build + SDL3 linkage.
    - Import the SDL3 module as `const c = @import("c");` (name matching build.zig translator.mod).

    - Single line: `pub const Emulator = @import("Emulator.zig").Emulator;`

      - Fields: `allocator: std.mem.Allocator`

      - `pub fn deinit(self: *Emulator) void` — no-op body.
